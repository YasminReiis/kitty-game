🐱 Kitty Run 💖

Este é um jogo simples desenvolvido apenas como um projeto leve e criativo, com foco em diversão e uma pequena surpresa no final.

A ideia foi inspirada no clássico jogo do dinossauro offline, porém sem a intenção de criar algo difícil ou complexo. O objetivo é apenas pular obstáculos até atingir a pontuação de 1000 pontos.

Ao chegar nesse objetivo, o jogo revela uma mensagem especial:
💖 "Ana, eu te amo"

O projeto funciona mais como uma mistura de brincadeira com uma pequena declaração, sem sistemas avançados ou mecânicas complexas. Não possui salvamento, progressão elaborada ou dificuldade alta — é apenas um passatempo simples feito para um momento específico.

🎮 Acesse o jogo

👉 https://yasminreiis.github.io/kitty-game/

✏️ Personalização

Caso queira usar esse projeto para outra pessoa ou adaptar a ideia, basta modificar a frase diretamente no código:

let message = "EU TE AMO ";

Você pode alterar para qualquer mensagem que quiser.

💡 Sobre o projeto

Este projeto foi feito apenas por diversão, como forma de praticar programação e criar algo diferente e personalizado. Não tem como objetivo ser um jogo completo, mas sim uma experiência simples e criativa.
